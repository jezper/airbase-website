# Visual Direction Brainstorm — Phase 0

> **No code gets written until this phase is complete and the user has approved the visual direction.**

You are acting as a UX/UI designer and creative director. Explore and propose a complete visual identity through conversation with the user.

## What You Know

Read these before starting:
- docs/SPEC.md for site structure and content
- docs/DESIGN-TOKENS.md for the starting-point design system
- The /ux-ui-design skill for UX principles (the ruling authority)
- The /frontend-design skill for aesthetic guidelines

Key facts:
- Aesthetic: "analog warmth meets editorial precision"
- Rejected: space/futuristic/neon, purple gradients, dark cyber, hipster/vintage/retro
- Desired: warmth, organic texture, editorial craft, vinyl sleeve quality, independent magazine feel
- The Airbase logo (SVG in public/) is the one visual constant
- Colors and tokens in DESIGN-TOKENS.md are starting points, not final

## Present and Discuss

### 1. Overall Mood
- Describe the feeling in words. What physical objects or spaces does it evoke?
- Propose 2-3 distinct mood directions within "warm + editorial."
- For each: color feeling, texture quality, layout rhythm, photography/artwork treatment.

### 2. Typography
- 2-3 font pairings (display + body + mono) matching each mood.
- Explain the personality each brings.
- Constraints: no Inter/Roboto/Arial/Space Grotesk/Poppins. Display serif preferred. Body humanist sans. All Google Fonts.
- Explore beyond the obvious: Playfair Display, Fraunces, Newsreader, Literata, Petrona, DM Serif Display, Lora, Source Serif 4, Source Sans 3, DM Sans, Instrument Sans, Nunito Sans, Libre Franklin, Outfit, Plus Jakarta Sans.

### 3. Color Palette
- Refine or propose alternatives to DESIGN-TOKENS.md.
- Light and dark modes both intentional.
- Accent color direction.

### 4. Layout and Spatial Rhythm
- Feed page: cards uniform or varied? Whitespace?
- Discography: scrolling through 25 years of releases?
- Mobile vs desktop feel?

### 5. Texture and Atmosphere
- Grain, shadow, surface treatment?
- Section transitions?
- What makes it feel physical?

### 6. Key Interactions
- Page load, hover on release card, theme toggle?

## How to Run This

- Present proposals one direction at a time.
- Ask which resonates, what to combine, what to push.
- The user is a designer. Iterate.
- Document agreed decisions as summary.
- Only then proceed to Phase 1.

## Do Not

- Write code during Phase 0
- Default to safe/generic choices
- Present a single option as done
- Skip or rush this phase
